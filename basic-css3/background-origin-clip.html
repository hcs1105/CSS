  • background-origin 속성

    배경의 원점을 엘리먼트의 어느 위치에 배치할지를 지정하는 속성입니다.

    background-attachment 속성값이 'fixed'인 경우, background-origin 속성은 적용되지 않습니다

    속성값들은 다음과 같습니다.

    • border-box : 배경을 테두리에 상대적으로 배치합니다.
    • padding-box : 배경을 내부 여백에 상대적으로 배치합니다(기본값).
    • content-box : 배경을 콘텐츠에 상대적으로 배치합니다.
  • background-clip 속성

    엘리먼트의 배경을 어디까지 차지할지를 지정하는 속성입니다.

    속성값들은 다음과 같습니다.

    • border-box : 배경이 테두리의 바깥 경계까지 차지합니다(기본값).
    • padding-box : 배경이 내부 여백의 바깥 경계까지 차지합니다.
    • content-box : 배경을 콘텐츠 상자에 맞춰 표시합니다.
    • text : 배경을 전경 텍스트 위에만 표시합니다.
