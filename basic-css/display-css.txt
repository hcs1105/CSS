[display]
1. 정의
엘리먼트를 어떻게 보여줄지를 지정하는 속성

2. 종류
① block
  - 엘리먼트를 블록(block) 엘리먼트로 만드는 속성
② inline
  - 엘리먼트를 인라인(inline) 엘리먼트로 만드는 속성
  - 엘리먼트의 폭과 높이를 지정할 수 없습니다.
  - 엘리먼트의 좌우 여백만 지정할 수 있습니다.
③ inline-block
  - 엘리먼트를 인라인 블록(inline-block) 엘리먼트로 만드는 속성
  - 인라인(inline) 엘리먼트처럼 한 줄로 표시됩니다.
  - 블록(block) 엘리먼트처럼 모든 방향에 여백을 줄 수 있습니다.
  - 블록(block) 엘리먼트처럼 엘리먼트의 폭과 높이를 지정할 수 있습니다.
  - IE 7 이하에서 적용되지 않지만 css 핵(css hack)으로 해결이 가능합니다.
④ none
  - 화면에서 해당 엘리먼트가 사라집니다.
  - 화면 낭독기가 읽지 않습니다.
